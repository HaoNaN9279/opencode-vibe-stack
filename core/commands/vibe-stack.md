## Vibe Stack Commands

Use the `vibe-stack` CLI for managing domain configurations.

### Available commands:
- `list` — List all available domains
- `status` — Show active domains in current project
- `activate <domain>` — Activate a domain for the current project
- `deactivate <domain>` — Remove a domain from the current project
- `use-stack <stack-name>` — Activate a preset domain stack
- `core-update` — Re-sync core config symlinks
- `help` — Show help

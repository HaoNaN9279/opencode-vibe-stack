## What I do

分析代码的结构和问题，制定重构计划，逐步执行重构并验证功能等价性。

## When to use me

当你需要优化函数结构、提升代码清晰度、或按照最佳实践整理现有代码时使用。

## Steps

1. **Analyze** — 分析代码的结构、问题和改进点
2. **Plan** — 给出重构计划，说明每个步骤的目的
3. **Execute** — 实现重构后的代码，保持功能完全一致
4. **Verify** — 验证重构后的代码与原代码功能等价

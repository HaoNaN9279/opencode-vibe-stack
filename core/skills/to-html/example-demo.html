📖 项目简介

mermaid-cli（包名 @mermaid-js/mermaid-cli）是 Mermaid 官方推出的命令行工具。 它接收一个 Mermaid 定义文件（.mmd）作为输入，通过 Puppeteer 无头浏览器渲染， 输出为 SVG、PNG 或 PDF 格式，是文档自动化、CI/CD 管线中嵌入图表的首选工具。

✨ 核心特性

🎯 多格式输出

  • SVG 矢量图（可动画）
  • PNG 位图（自定义背景/缩放）
  • PDF 文档输出

🎨 主题与样式

  • 支持暗色/亮色主题
  • 自定义 CSS 注入
  • 透明背景选项
  • Iconify / FontAwesome 图标支持

📄 Markdown 处理

  • 直接处理含 Mermaid 代码块的 .md 文件
  • 自动提取 → 渲染 → 替换为图片引用
  • Docker / CI 友好

📦 安装方式

npm 全局安装

npm install -g @mermaid-js/mermaid-cli

npx 免安装运行

npx -p @mermaid-js/mermaid-cli mmdc -h

Docker

docker run --rm -v $PWD:/data \
  ghcr.io/mermaid-js/mermaid-cli \
  mmdc -i /data/input.mmd \
       -o /data/output.svg

Homebrew (macOS)

brew install mermaid-cli

⚡ 快速使用

基本用法 — .mmd → SVG

mmdc -i input.mmd -o output.svg
更多示例

生成 PNG（暗色主题 + 透明背景）

mmdc -i input.mmd -o output.png \
  -b transparent \
  --backgroundColor transparent \
  --theme dark

处理 Markdown 文件

mmdc -i README.md -o README-out.md

管道输入（stdin）

mmdc -o output.svg < B
EOF

限制并行渲染数

mmdc -i input.mmd -o output.svg --jobs 4

🔌 Node.js API

mermaid-cli 也提供了 JavaScript API，可在 Node.js 中以编程方式调用：

import { run, renderMermaid } from '@mermaid-js/mermaid-cli';

// 方法 1：文件渲染
await run('input.mmd', 'output.svg');

// 方法 2：渲染 Mermaid 定义字符串
const svg = await renderMermaid('mermaid-code', {
  backgroundColor: '#fff',
});

⚠️ Node.js API 不遵循 semver，版本跟随 mermaid 主版本。
